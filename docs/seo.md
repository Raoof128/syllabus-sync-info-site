# SEO and machine discoverability

- Unique titles and descriptions are generated for every content route.
- Canonicals resolve against `https://info.syllabusing.app`.
- The homepage contains visible, server-rendered copy and `WebSite` JSON-LD.
- `sitemap.xml`, `robots.txt`, `llms.txt`, Open Graph media and a web manifest are generated by Next.js.
- Normal search crawlers and OAI-SearchBot are allowed. GPTBot is denied to keep search discovery separate from model-training access.
- FAQ content is visible but does not use `FAQPage` schema.
- No rating, review, organisation relationship or software-offer schema is fabricated.

Search Console, Rich Results Test and deployed URL Inspection remain launch-time external checks.
